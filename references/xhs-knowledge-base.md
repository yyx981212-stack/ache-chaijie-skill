# 小红书账号拆解知识库

正式分析账号时读取。官方规范的优先级高于运营书籍和经验方法；平台规则可能变化，对涉及商业合作、高风险品类或当前政策的结论，再打开官方页核对版本和更新时间。

## 官方规范基线

以小红书官方《社区公约2.0》、《社区规范》及蒲公英帮助中心的当前规则为依据，将账号策略审计为三层：真诚分享、友好互动、有序经营。

| 官方原则 | 分析账号时检查 | 策略判断 |
|---|---|---|
| 原创、真实、可核实 | 素材是否来自真实经验；案例、数据、身份和效果是否有证据 | 真实过程、细节和局限性声明是信任策略，不是无效冗余 |
| AI 辅助主动标明 | 文案、图片、视频或润色是否明确说明 AI 参与 | 对 AI 账号记录标注方式；未标注时只写“存在合规检查项”，不断言违规 |
| 文题相符，不用夸张、猎奇骗点击 | 标题和封面承诺是否在内容中兑现 | 高点击包装可复制，但“承诺大于证据”必须列为风险 |
| 不塑造虚假人设 | 财富、学历、收入、经历、生活场景是否可验证 | 人设可编辑表达，不能建立在虚构资历和摆拍生活上 |
| 科学、资质与安全 | 健康、医疗、投资、购房等建议是否有资质和科学依据 | 缺少资质的“必然有效”叙事不列为可复制打法 |
| 友好互动 | 是否使用性别、地域、职业、阶层对立或未经证实的指控制造讨论 | 区分有信息价值的争议与恶意引战，不把后者推荐为爆款模板 |
| 亮明经营身份 | 品牌、经销商、买手、店主或有利益关系的推荐是否透明 | “伪装素人”即使互动高，也不得归入可复制策略 |
| 不虚构功效、不伪造数据 | 功效、销量、评价、点赞收藏和收益承诺是否有支撑 | 夸大功效、保证收益、绝对化承诺和刷量列为商业风险 |
| 平台内有序经营 | 是否引导站外或线下交易；是否滥发广告和私信骚扰 | 分析变现时先识别官方平台承接，不把规避平台的导流话术当作运营技巧 |
| 知识产权与隐私 | 是否搬运图片、文案、视频，或未授权使用肖像和个人信息 | 学习结构不等于复制素材；输出“可迁移规则”而非让用户照搬 |

### 商业合作附加审计

内容合作笔记执行先审后发，审核通路包含安全、内容和客户审核；叠加广告投放时还需广告审核。分析商单笔记时增加：

- 是否存在保证性、绝对化或不符合常识的效果承诺。
- 是否把普通食品、化妆品或日用品与医疗、保健功效不当关联。
- 是否有真实使用场景、可核实证据和局限性，而不只是官方宣传话术。
- 是否贬低竞品、伪造评价、干预差评或伪装普通用户口碑。
- 是否涉及禁入、特殊准入或需专业资质的行业；此类必须打开当前官方规则复核。

### 官方规范与“爆款重复”

重复已验证的人群需求、选题母题和视觉识别不等于违规。但每篇仍应有真实的新场景、新案例或新信息，并保持文题相符、素材合法、身份透明。批量、高频、机器式发布，搬运素材，虚假口碑或伪造数据是另一类问题，不得与有效复用混为一谈。

## 基础判断

1. **定位**：用“谁在什么场景下，为哪类人解决什么问题，凭什么值得信任”表达，不只写赛道名。
2. **账号承接**：内容获得单次点击，昵称、头像、简介、置顶和系列内容共同提供持续关注理由。
3. **内容价值**：封面和标题负责表达人群、问题和结果；正文或视频必须用步骤、清单、对比、过程或成果物兑现承诺。
4. **爆款复用**：优先重复已验证的用户需求、选题母题和表达结构，同时更换场景、案例、商品、时间或证据。
5. **数据复盘**：用多篇同母题样本寻找稳定规律；不用单篇爆款概括整个账号，不用点赞代替转化证据。
6. **品类适配**：将平台共性与品类的视觉、搜索、信任、陪伴或购买决策特征结合，不跨赛道机械套模板。

## 知识库边界

- 书中的“爆款都是重复的”可作为内容复盘启发，但不是小红书官方算法说明。
- 不把书中关于权重、流量池、推荐阈值或用户规模的时效性数字写入分析结论。
- 将官方社区规范、创作规范和商业合作规则视为更高优先级；高风险结论必须核对当前官方版本。
- 只有账号的公开样本能证明该账号的重复策略是否有效；书中方法不能替代实际数据。
